TPHXParticleRenderer = procedure(Particles: TPHXParticleList; Canvas: TPHXCanvas; Buffer: TPHXBuffer);

Render callback used to render a list of particles

Parameters
Particles
List of parameters to render
Canvas
The canvas to render to for 2D particles
Buffer
The canvas to render to for 3D particles
TPHXEmissionMode = (...);

Emission mode of an particle emittor

Values
  • emTime: Emit particles over time
  • emDistance: Emit particles over distance traveled
TPHXParticlePatternSize = (...);

Size of the pattterns for a particle, for speed only rectangular patterns is suppported

Values
  • ps1x1: No pattern, use the full texture
  • ps2x2: 4 patterns, arranged in 2 rows and 2 columns
  • ps4x4: 16 patterns arranged in 4 rows and 4 columns
  • ps8x8: 64 patterns, arranged in 8 rows and 8 columns
TPHXParticleBlending = (...);

Blending for each particle system

Values
  • bmNone: Normal, none blending
  • bmAdd: Additive blending
  • bmSub: Subtractive blending
  • bmAlpha: Alpha blending
TPHXParticleShape = (...);

Shape of the particles, determines the particle renderer to use

Values
  • psPoint: The particle is rendered as a single point
  • psBillboard: The particle is rendered facing the camera
  • psBillboardRotated: The particle is rendered facing the camera with spin
  • psAlignX: The particle is rendered as a quad with the normal pointing in the X-Axis
  • psAlignXRotated: The particle is rendered as a quad with the normal pointing in the X-Axis and with particle spin
  • psAlignY: The particle is rendered as a quad with the normal pointing in the Y-Axis
  • psAlignYRotated: The particle is rendered as a quad with the normal pointing in the Y-Axis and with particle spin
  • psAlignZ: The particle is rendered as a quad with the normal pointing in the Z-Axis, the default for 2D particle systems
  • psAlignZRotated: The particle is rendered as a quad with the normal pointing in the Z-Axis and with particle spin
  • psTrail: The particle is rendered as interconnected, billboarded quads
  • psTrailZ: The particle is rendered as interconnected axis aligned quads
  • psCustom: Custom particle shape, you have to register a renderer for this shape with RegisterRenderer
