Supported pixel formats

Values
  • pfNone: Empty pixel format
  • pfAlpha: 8 bit image containing grayscale values pfGray, 8 bit image containing alpha values
  • pfRGB: 24 bit
  • pfRGBA: 32 bit
TPHXPixelChannel = (...);

  • pcBlue: Blue channel
