Types

TPHXTextAlign = (...);

Text alignment for rectangles

Values
  • taTopLeft: Align to the top left of the rectangle
  • taTop: Align to the top
  • taTopRight: Align to the top right of the rectangle
  • taLeft: Align to the left of the rectangle
  • taCenter: Align to the center of the rectangle
  • taRight: Align to the right of the rectangle
  • taBottomLeft: Align to the bottom left of the rectangle
  • taBottom: Align to the bottom of the rectangle
  • taBottomRight: Align to the bottom right of the rectangle
TPHXAlignment = (...);

Text alignment for a single line

Values
  • alLeft: Align the text to the left
  • alCenter: Align the text in the center
  • alRight: Align the text to the right
TPHXFontStyle = (...);

Font style

Values
  • fsBold: Bold
  • fsItalic: Italic
  • fsUnderline: Underline
  • fsStrikeOut: Strikeout
TPHXFontStyles = set of TPHXFontStyle;

Set of font styles

TPHXFontWrap = (...);

Wrapping mode for fonts

Values
  • wmWord: Word wrap for each word
  • wmChar: Word wrap on a character for character basis
