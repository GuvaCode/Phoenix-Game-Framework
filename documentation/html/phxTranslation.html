Unit phxTranslation

DescriptionUsesClasses, Interfaces, Objects and RecordsFunctions and ProceduresTypesConstantsVariables

Description

Add support for localisation your project

A example localisation file can look like this 

    -- Commented line
    name = English
    code = en_US

    welcome = Welcome\nto this application

This file can be loaded with the TPHXTranslationManager.Load function.

To translate a tag call the TPHXTranslationManager.Translate function or one of its overloads after the translation file is loaded.

The default behaviour when translation a missing tag is to return the tag name to raise a exception instead the TAG_EXCEPTION define to the project.
