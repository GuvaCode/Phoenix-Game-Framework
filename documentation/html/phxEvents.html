Event types

Values
  • PHX_EVENT_NONE: No event
  • PHX_EVENT_QUIT: The user has requested to quit the application
  • PHX_EVENT_USER: User generated event
  • PHX_DEVICE_RESIZED: TPHXDeviceEvent The device window has been resized
  • PHX_DEVICE_RESET: Occurs after a device is reset, allowing an application to re-create all resources.
  • PHX_DEVICE_ACTIVATED: Occurs if the device is activated, ie gaining focus
  • PHX_DEVICE_DEACTIVATED: Occurs if the device is deactivated, ie lost focus
  • PHX_MOUSE_PRESSED: TPHXMouseButtonEvent A mouse button has been presssed
  • PHX_MOUSE_RELEASED: A mouse button has been released
  • PHX_MOUSE_MOVED: The mouse has been moved,
  • PHX_MOUSE_WHEELUP: Mouse wheel up
  • PHX_MOUSE_WHEELDOWN: Mouse wheel down
  • PHX_KEY_PRESSED: TPHXKeyboardEvent A keyboard key has been pressed,
  • PHX_KEY_RELEASED: A keyboard key has been released,
  • PHX_KEY_CHARACTER: Event for keyboard text input
